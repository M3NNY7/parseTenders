<h1>This script for parsing tenders from bidzaar!</h1>

<p>for start parser, first must install all requirements from file requirements.txt, then in file that you want start, replace strings cookies, headers, with your cookies and headers from site bidzaar. Copy your cookies with generate curl request in browser from devtels, and go to site<a href="https://curlconverter.com">curlconverter</a> and generate request for python/request. Then, copy request that you generated, and past instead of cookies, headers, and params</p>
<p>After all, when you installed all requirements, and pasted cookies, and headers to files that you want start, use default command:</br> <b>python3 bidzaar_shops.py<b/></p>
